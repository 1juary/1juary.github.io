## 🚀 Quick Start

```bash
git clone https://github.com/1juary/1juary.github.io.git
cd 1juary.github.io
bundle install
bash run_server.sh
```

Then open [http://127.0.0.1:4000](http://127.0.0.1:4000) 🌐

## 📁 Project Structure


## 📝 Customization

1. ✏️ Edit [`_pages/about.md`](_pages/about.md) to update your bio and experience

3. ⚙️ Update [`_config.yml`](_config.yml) with your info
